JSON

发表于 2016-04-29   |   分类于 JavaScript

三种类型

简单值

可以表示字符串、数值、布尔值和null，但不支持undefined
表示字符串必须用双引号

对象

  与javascript中稍有不同
  javascript中：
      var person = {
           name:"Nicholas",
           age:29
         };
  JSON中：
      {
           "name":"Nicholas",
           "age":29
      }    
三点不同：
  1.没有声明变量（没有变量概念） 
  2.没有末尾分号
  3.属性要求加双引号（其实javascript也可以加）
