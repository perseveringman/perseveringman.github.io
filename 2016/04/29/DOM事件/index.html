DOM事件

发表于 2016-04-29   |   分类于 JavaScript

什么是DOM事件？

点击一个DOM元素
键盘按下一个键
输入框输入一个内容
页面加载完成
...

事件流

事件捕获   capture phase
      从window开始向下捕获，一直到事件发生的标签的父元素
事件触发   target phase
      目标节点的事件触发
事件冒泡   bubble phase
      从事件发生的父元素开始冒泡到window对象

IE低版本没有捕获过程
页面加载没有冒泡过程
只要在整个事件流的过程中的对象上注册，就会发生

事件注册与触发

事件注册

 1.eventTarget.addEventListener(type,listener[,useCapture(事件是否是捕获过程)])
eg:
var elem = document.ElementById('div')
 var clickHandler = function(event){
     //...
}
elem.addEventListener('click',clickHandler,false);

2. elem.onclick = clickHandler;  //这种方法只能注册一个函数

事件取消注册

eventTarget.removeEventListener(type,listener[,useCapture])

eg:
elem.removeEventListener('click',clickHandler,false);
elem.onclick = null;//不建议

事件触发

eventTarget.dispatchEvent(type)
elem.dispatchEvent('click');

浏览器兼容（IE6,7,8）

事件注册与取消
      attachEvent/detachEvent
事件触发
      fireEvent(e)
no capture(捕获)

兼容代码

var addEvent ＝ document.addEventListener ?
      function(elem,type,listener,useCapture){
            elem.addEventListener(type,listener,useCapture);
      }:
     function(elem,type,listener,useCapture){
          elem.attachEvent('on' + type,listener);
      };
delEvent类似

事件对象

var clickHandler = function(event){}   //event就是事件对象

IE低版本需要在函数里先写event ＝ event || window.event;

属性
- type   //事件类型
- target(srcElement浏览器兼容)  //元素
- currentTarget当前处理事件的函数，前面说道事件并不一定注册到元素，有可能是父元素等

方法
- stopPropagation //阻止事件传播
       event.stopPropagation()(w3c)
       event.cancelBubble = true(IE)
       event.stopImmediatePropagation()(w3c) //还阻止后续事件
- preventDefault//阻止默认行为,如点击链接不打开页面
       event.preventDefault()(w3c)
       event.returnValue = fasle (IE)

事件类型

鼠标事件

click
dbclick
mousedown  按下鼠标
mousemove
mouseover     移上
mouseout     移开
mouseup
mouseenter   不冒泡   进入元素
mouseleave     不冒泡   移开元素

鼠标事件对象

属性
clientX，Y  //到页面的左端和顶端
screenX，Y // 到电脑屏幕的左端和顶端
button（0，1，2）左中右键
ctrlKey......//对应ctrl键等，按下为true

鼠标事件顺序

从元素A上方移过
    mousemove－》mouseover（A）－》mouseenter（A）－》很多mousemove（A）
    －》mouseout（A）－》mouseleave（A）
点击元素
       mousedown－》［mousemove］移动会触发－》mouse－》click
