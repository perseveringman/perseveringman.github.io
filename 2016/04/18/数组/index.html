数组

发表于 2016-04-18   |   分类于 JavaScript

tips

弱类型，可以含有不同类型的元素、对象、数组。
可以动态添加数组元素
delete可以删除元素，但长度不变只是变为undefined

定义声明

var arr = [];
var arr = [,,]
//数组允许最后有逗号，这里返回undefined*2
var arr = new Array(); 
var arr =  new Array(100); //undefined*100
var arr = new Array('a','b');//赋值

一些属性

arr.push(1);

arr[arr.length]=4;//小技巧

arr.unshift(0);//在头部添加元素

arr.length-=1;//删除尾部元素

arr.pop();//同上

arr.shift();//头部删除元素

数组迭代

想要用for...in的话，可以这样写：
 for(i in arr){
     if(arr.hasOwnProperty(i))
    {console.log(arr[i];)}
}

稀疏数组

var arr1=new Array(100);
var arr1[99]=1;
99 in arr1;//true
98 in arr1;//false

数组方法

Array.prototype.join数组转为字符串

 var arr =[1,2,3];
 arr.join();//"1,2,3"
 arr.join("_");//"1_2_3"
 function repeatString(str,n){
   return new Array(n+1).join(str);
}//重复某个字符串
 repeatString("a",3);//"aaa"

Array.prototype. reverse数组逆序

arr.reverse();//[3,2,1]
当调用之后，数组被修改

Array.prototype.sort()数组排序

 按字母顺序排序
 想按数字排序：
 arr.sort(function(a,b){return a-b;});
当调用之后，数组被修改

Array.prototype.concat 数组合并

arr.concat(4,5);//[1,2,3,4,5]
数组未被修改

arr.concat([4,5],6);//[1,2,3,4,5,6]
数组被拉平了！

Array.prototype.slice返回部分数组

var arr1=[1,2,3,4,5];
arr1.slice(1,3);//[2,3]
arr.slice(1);//[2,3,4,5]
arr.slice(1,-1);//[2,3,4]
arr.slice(-4,-3);//[2]
第一个参数表示从哪个开始，第二个表示到结束的下一个，支持负数，表示从后往前，数组未被修改

Array.prototype.splice 数组拼接

arr1.splice(2);//[3,4,5];
arr1；//[1,2]
数组被 修改相当于减去那部分
arr1.splice(2,2);//[3,4]第二个参数表示个数
arr1.splice(1,1,'a','b');//2第三个之后的参数表示要加入的元素
