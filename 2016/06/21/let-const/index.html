let&const

发表于 2016-06-21   |   分类于 JavaScript

let

1.块级作用域

  1. 没有变量提升、且存在暂时性死区（temporal dead zone，简称TDZ）
    tdz区域使用该变量会报错

    if (true) {
    // TDZ开始
    tmp = ‘abc’; // ReferenceError
    console.log(tmp); // ReferenceError

    let tmp; // TDZ结束
    console.log(tmp); // undefined

    tmp = 123;
    console.log(tmp); // 123

    }

const

1.声明只读常量（之后赋值全部无效，所以必须立即赋值）
2.常量不提升，暂时性死区
3.赋值为对象类型时（数组之类的也算），可以继续添加属性。
4.可以使用Object.freeze({})冻结对象，这样就不能继续添加属性
