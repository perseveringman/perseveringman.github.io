node.js基础—npm

发表于 2016-06-29   |   分类于 JavaScript

npm

start a project

package.json

npm init —scope=<username>

npm help json查看package.json的帮助文档

install a module

npm install <modulename>  安装特定包

listing dependencies

npm ls   查看当前的包

在安装时 输入npm install –save
如果不输入这一条，在npm ls时会出现npm ERR! extraneous:。。。
官方解释是： 

Your dependencies should be listed in the package.json file in an object called 'dependencies'.  However, when we installed '@linclark/pkg', we didn't update the package.json file to list out this dependency.

就是没有更新package.json,使用—save能避免这个问题

npm Test

npm 需要有test.js文件，并且在package.json中还得修改
“scripts”: {
“test”:”node test.js”
}

version

版本会随着更新的发布而改变，具体见下图

npm version 1.0.1可手动更改版本号

过期检查

npm outdated

检查是否有过期的包

删除

npm rm —save 删除所有
npm uninstall <>删除一个
